## Prerequisites

- Running docker
- Running postgres-db (maybe also on docker) (see also [Postgres-Docker-compose](https://github.com/MaxWenzel/docker-compose/tree/master/postgres))

## Usage

```bash
http://localhost:8082/postalcodes
```
